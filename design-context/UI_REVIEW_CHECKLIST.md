# UI Review Checklist

## Product Fit

- Does the UI match the product character?
- Does it feel specific to this product?
- Could this screen belong to any other SaaS app?

## Visual Hierarchy

- Is the primary action obvious?
- Is typography doing enough work?
- Are sections clearly grouped?

## Distinctiveness

- What is the signature design moment?
- Which generic pattern was avoided?
- Is there a memorable layout or interaction?

## Consistency

- Are spacing, radius, color and typography coherent?
- Are components reused intentionally?
