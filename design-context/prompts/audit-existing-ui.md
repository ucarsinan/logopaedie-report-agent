# Audit Existing UI

Audit the current UI for generic AI-generated design patterns.

Identify:

1. Generic patterns
2. Visual inconsistencies
3. Missing product character
4. Weak typography
5. Repetitive layout patterns
6. Components that feel template-like
7. Opportunities for signature design moments

Then propose a phased redesign:

- minimal improvements
- medium-impact changes
- bold distinctive changes
